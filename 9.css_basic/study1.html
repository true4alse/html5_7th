css

html태그를 꾸며주는 언어

기초문법

css는 태그를 선택해서 스타일을 입력하는 언어. 태그를 선택하는 부분과 스타일이 입력되는 부분으로 나뉜다.

기초문법 - 선택자{스타일속성:스타일속성값;스타일속성:스타일속성값;}

선택자는 기본적으로 복수의 개념

  • 태그명 -> 태그명만 선택자로 기입
  • 클래스명(중복사용가능) -> 접근(문)자 .을 사용
  • 아이디명(중복사용불가능) -> 접근자 #을 사용

;(세미콜론)은 스타일속성을 나누는 구분자로 사용된다

스타일 입력 방식

  • 인라인스타일 - 태그에 직접 스타일속성을 사용
  • 내부스타일시트 - style태그를 만들어서 사용
  • 외부스타일시트 - style파일을 만들어서 연결하여 사용

인라인스타일 단점

태그를 일일이 선택하여 시작태그에 style속성을 사용하는 방식이기 때문에, 일괄 수정이 안된다.

내부스타일시트방식 단점

스타일시트가 html파일안에 포함되어있기 때문에 여러 페이지를 동시에 일괄수정할 수 없다.
