함수 - function

자바스크립트에서 동작을 표현하는 데이터 형태

함수는 기본적으로 정의가 되어야 실행할 수 있다

alert,prompt,typeof,Number 와 같은 함수들은 기본적으로 브라우저에 이미 정의가 되어있기 때문에(표준내장함수), 정의하지 않은 상태에서 함수를 실행할 수 있었다

사용자 정의함수

등록하는 기본형태 - function 등록할함수명(재료투입구매개변수){동작할 소스코드}

사용형태 - 등록한함수명(재료인자)

  1. 함수는 정의를 먼저 해야 함
  2. 정의한 함수를 실행

함수 안에서 return 표현

return키워드는 필수적으로 사용하는 것이 아니고 선택적으로 필요에 따라서 함수안에서 사용할 수 있다.

함수안에서 return구문을 만나면 그 즉시 함수가 종료가 되고 return키워드 우측에 있는 값을 함수가 사용되었던 자리에 남기게 된다
