float

블록 태그들을 좌우로 나란히 배치하는데 목적이 있다.

선택한 태그를 둥~~~ 띄워서 부모의 컨텐트박스 안에서 왼쪽 혹은 오른쪽으로 배치하는 방법

플로팅된 자식태그들은 집나간 자식태그가 된다 -> 부모태그가 자식태그로 인식하지 않는다. ->부모태그가 자식태그를 포함할만큼 높이를 자동으로 측정할 수 없다.

clear

float를 해제할 때 사용하는 속성

clear속성이 적용된 태그는 앞서 만들어진 형제태그 중에서 float되어있는 태그 중 가장 아래까지 내려와있는 태그의 바로 밑에 배치되는 특징을 가지고 있다.
