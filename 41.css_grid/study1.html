그리드 컨테이너에 지정하는 속성들

display:grid

display:grid속성이 지정된 태그는 그리드 컨테이너가 된다. 해당 태그의 자식태그들은 그리드 아이템이 된다.

grid-template-columns

열의 개수와 크기를 지정할 수 있는 css속성

기본적인 단위는 px을 사용하며 %단위를 쓸 수도 있지만 %단위보다는 fr(fraction)단위를 더 많이 사용한다.

repeat(열의개수,지정할 크기)함수를 사용하여 열의 개수와 크기를 쉽게 지정할 수 있다

grid-template-rows

행의 개수와 크기를 지정할 수 있는 css속성

grid-auto-rows

열의 개수에 따라 행의 개수가 달라지기 때문에 자동으로 행의 개수와 크기를 지정하는 편이 좋다.

minmax(최소값,최대값)함수를 사용하여 보다 유동적인 레이아웃을 구성할 수 있다.

justify-content

메인축(가로축)을따라서 전체 그리드컨테이너 공간안에서 아이템그룹을 정렬하는 방식

기본값 : start / (start,end,center,space-between)

justify-items

각각의 그리드 아이템(셀)기준으로 안쪽의 컨텐츠 위치를 가로축을따라서 정렬하는 방식

기본값: stretch / (start,end,center)

align-content

교차축(세로축)을 따라서 전체 그리드 컨테이너 공간안에서 아이템그룹을 정렬

기본값 : start / (start,end,center,space-between)

align-items

각각의 그리드 아이템(셀)기준으로 안쪽의 컨텐츠 위치를 세로축을 따라 정렬

기본값: stretch / (start,end,center)

gap

gap:row값 column값 을 사용하여 각각의 그리드 셀사이에 여백을 만들어 줄 수 있다

그리드아이템에 지정하는 속성들

justify-self / align-self

이속성들은 그리드 컨테이너에서 지정했었던 justify-items/align-items 속성과 결과는 같다. 하지만 적용대상이 그리드아이템이라는 것에 차이가 있다.
