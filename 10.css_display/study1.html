html은 먼저만든 태그가 블록처럼 쌓이는 구조이다. -거꾸로하는 테트리스-

태그의
형태 - display

  • display:block -> 블록레벨태그 - h1,h2,h3...,p,ul,ol,li,dl,dt,dd,div,header,footer...
  • display:inline -> 인라인레벨태그 - a,b,strong,i,em,label
  • display:inline-block -> 인라인블록레벨태그 - img,video...input type=submit

블록레벨태그 특징

  • body태그html태그도 블록레벨태그이고, 사용자가 브라우저의 너비를 줄이고 늘릴 때 자동으로 조절되는 기본값 (width:auto)가 설정되어 있다.
  • 블록레벨태그의 높이는 포함하는 컨텐츠의 양만큼 늘어나거나 줄어드는 auto값이 기본값이다.
  • 포함하는 컨텐츠의 양과 관계없이 부모만큼의 너비를 상속받는다
  • 박스형태를 가지고 있다 -> 가로와 세로사이즈의 조절이 가능하다

인라인레벨태그 특징

  • 포함하는 컨텐츠의 양만큼의 너비를 차지한다 -> 나란히 글자처럼 배치된다
  • 박스형태가 아님 -> 가로와 세로사이즈를 조절할 수 없다

인라인블록레벨태그 특징

  • 인라인레벨태그처럼 베이스라인을 기준으로 배치가 되고, 블록레벨태그처럼 박스형태를 가지고있는 형태
  • 가로세로사이즈 조절이 가능하다.
  • 줄바꿈이 안되고 나란히 배치가 된다.
