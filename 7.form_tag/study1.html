b

홈페이지를 사용하는 사용자가 내용을 입력할 수 있도록 미리 제공하는 양식

label태그는 입력박스에 대한 이름표태그이다.

label태그와 input태그를 서로 연결하기 위해서는 input태그의 id속성과 label태그의 for속성을 똑같이 적용하면 된다.

label태그가 눈에보이지 않아도 구성해야하며 css에서 안보이게 처리하는 방법을 사용한다.

input태그의 속성이 radio인 경우 name속성 값을 동일하게 적용하면 여러개의 radio타입 버튼중 1개만 선택되도록 할 수 있다.

input태그 중 type속성이 text로 적용되어있다면 짧은 한줄의 텍스트를 입력받을 용도로 사용하고, textarea태그는 장문의 텍스트를 입력 받을 때 사용한다.

form태그 안에서도 문맥태그를 자유롭게 사용할 수 있다.

사용되는 속성

  • value - 입력박스에 들어가는 실제정보
  • disabled - 선택한 입력박스를 비활성 처리
  • required - 선택한 입력박스의 실제정보를 필수적으로 입력받게 할 때 사용
  • selected - option태그가 선택된 상태로 페이지가 로딩되도록 할 때 사용
  • checked - type속성이 radio/checkbox인 태그가 체크된 상태로 로딩되도록 할 때 사용
계정정보
  • 아이디는 영소문자 (한글X)
  • 아이디는 특수문자를 사용하시거나 띄어쓰기를 하시면 안됩니다.


비밀번호 입력규칙

  • 영 소문자 대문자 조합의 8~16자
  • 특수문자 최소 1개 사용
  • 띄어쓰기 사용 금지
