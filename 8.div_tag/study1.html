영역구성태그 - div

영역을 나눌 때 자유롭게 사용하는 태그

컨텐츠의 맥락에 따라서 영역을 자유롭게 나눌 때 사용

좌우로 나란히 배치되어있는 영역은 영역태그로 나누면 좋다

영역태그로 부터 파생되어 나온 태그들

  • header - 홈페이지의 머릿글 컨텐츠를 만들 때 사용(로고,주메뉴)
  • footer - 홈페이지의 꼬리말 컨텐츠를 만들 때 사용(하단로고,회사정보,운영정책 등)
  • main - 홈페이지의 전반적인 메인컨텐츠를 다룰 때 사용(main태그는 body태그안에 1개만 사용해야 함)
  • section/article - 영역태그와 같음.
  • nav - 홈페이지의 (2개이상의 다른 메뉴)메뉴단을 만들 때 사용
  • figure - 이미지 전용 영역태그(figcaption태그를 통해 삽화에대한 정보를 표기할 수 있다)
  • aside - 홈페이지 안에서 독립적인 사이드 컨텐츠를 만들 때 사용(전체 컨텐츠 공간안에 넣지않는것이 좋다.)
