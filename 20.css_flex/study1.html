flex

블록레벨태그들을 나란히 배치하는데 사용하는 레이아웃

구조

상하관계가 뚜렷 -> 대장(flex-container),쫄병(flex-item)

flex-container(대장)에 지정하는 속성 들

  • display:flex

    선택한 태그를 flex-container로 지정한다.

    flex-container는 (좌에서 우 방향-row)으로 메인축이 형성되고, 그에 수직방향(위에서 아래로-column)으로 교차축이 형성된다.

    flex-container의 자식태그들은 flex-item으로 분류되어 기본적으로 메인축을 따라서 나란히 정렬하는 특징을 갖게된다.

  • flex-direction

    메인축의 방향을 바꾸어 주는 속성

    • row(기본값) - 좌->우
    • row-reverse - 우->좌
    • column - 위->아래
    • column-reverse - 아래->위

    point)플렉스아이템들은 무조건 메인축을 따라서 나란히 배치된다.

    flex-start/center/flex-end 지점으로 방향성을 표시한다.

  • justify-content

    메인축을 따라 아이템을 정렬할 때 사용

    • flex-start(기본값) - 메인축 시작지점에 아이템 정렬
    • flex-end - 메인축 마지막지점에 아이템 정렬
    • center - 메인축 중앙지점에 아이템 정렬
    • space-between - 메인축 시작과 마지막지점에 아이템을 맞춘 후 아이템의 사이 여백을 동일하게 지정하도록 정렬
    • space-around - 각 아이템들의 좌우 여백을 동일하게 지정하여 정렬

  • align-items

    교차축을 기준으로 아이템을 정렬하는 속성

    기본값(stretch) 컨테이너의 교차축길이만큼 아이템을 늘림

    교차축의 방향을 설정하는 css속성 없음

    교차축은 메인축의 방향에 따라 결정되며 언제나 왼쪽에서 오른쪽 / 위에서 아래 두가지경우 밖에 없음

  • flex-wrap

    flex-item들의 줄바꿈을 설정하는 속성

    기본값(nowrap)줄바꿈을 허용하지 않음
    이 때는, 아이템들의 크기가 유동적임.
    박스사이즈가 자동계산

    wrap(줄바꿈을 허용)

  • align-content

    flex-container에서 아이템들의 줄바꿈을 허용했을 때 container전체 높이를 수직축으로하여 아이템 그룹 전체를 정렬할 때 사용하는 속성

flex-item 전용 속성들

아이템들은 포함하는 컨텐츠 만큼의 너비를 갖는다

  • flex-basis

    아이템들이 점유하는 기본크기 (auto:기본값)/width처럼 사용할 수 있다

  • flex-shrink

    아이템의 크기가 줄어드는 비율

    플렉스 컨테이너의 너비가 줄어들어서 아이템들에게 설정된 너비도다 작아져야할 때,줄어드는 비율(기본값:1)(아이템들이 줄바꿈이 허용이 안되는 상황에서 기본값1)

  • flex-grow

    플렉스 컨테이너의 공간이 여유가 있어서 남았을 때, 남은 공간을 아이템들이 나눠갖는비율(아이템은 커짐)

  • order

    아이템의 순서를 바꿀 때 사용

    order낮을 수록 축의 start지점에 정렬

  • align-self

    아이템의 수직축 정렬
